A collection of minimal Linux systems programming tools built in C, developed as part of EPL451.
Covers ELF binary analysis, syscall tracing, and ptrace-based debugging.

## Tools

| Folder | Tool | Description |
|--------|------|-------------|
| `STRACE/` | minimal_strace | ptrace-based syscall tracer |
| `MDB/` | minimal_gdb | GDB-style debugger with breakpoints, single-step, and disassembly |
| `FILE_LELF/` | minimal_file_lelf | `file`-command clone using libelf |
